# 🎨 乱涂彩世界 — 核心玩法循环深度拆解

> 换皮参考用：本文从「玩家时间尺度」拆解游戏机制，提炼爽点和设计逻辑。

---

## 一、游戏底层定位

- 官方自称：**副副副游** — 极轻量级，碎片化时间
- 核心公式：**单指操作 + 自动瞄准 + 弹幕染色 + 卡牌养成 + 三人联机**
- 一句话：一个披着"涂色"皮的竖屏弹幕射击卡牌手游

---

## 二、时间尺度拆解：玩家在不同时间段干什么？

### ⏱ 1分钟 — 单局战斗循环（核心爽点所在）

> **操作量：** 单指走位，自动攻击/染色
> **一局时长：** 约 90 秒

**玩家行为流：**

```
开局进场 → 单指滑动走位 → 自动攻击染色 → 积累乱涂等级
→ Lv5触发"乱涂爆发"（15%伤害+30%攻速移速） → 爆发输出
→ 针对BOSS弱点颜色攻击 → 弱点击破 → BOSS僵直 → 全力输出
→ 结算奖励
```

**每1分钟的爽点节奏：**

| 时间点 | 发生什么 | 爽点类型 |
|--------|---------|---------|
| 0-10秒 | 进场，开始喷射颜料染色 | 🎨 **视觉爽**：颜料喷溅全屏，黑白世界被染色彩色 |
| 10-30秒 | 乱涂等级逐渐提升，伤害增加 | 📈 **成长爽**：数字不断变大，感受变强 |
| 30-45秒 | 达到Lv5 → 爆发模式 | 💥 **爆发爽**：攻速移速暴增，全屏染色，视觉高潮 |
| 45-70秒 | 针对BOSS弱点攻击，弱点击破 | 🎯 **策略爽**：颜色克制打弱点，BOSS进入僵直 |
| 70-90秒 | BOSS击杀 / 结算 | 🏆 **收获爽**：掉落奖励，经验值增长 |

**为什么要短到90秒？**
- 通勤、排队、等电梯随时掏出来玩一局
- 高速正反馈循环，每局都有"爆发高潮"
- 输了也不难受，反正就90秒

---

### ⏱ 5分钟 — 日常循环（养成推进）

> **操作量：** 点几下，挂机为主
> **日常耗时：** 官方宣称「5分钟清日常」

**玩家行为流：**

```
登录领邮件/福利（30秒）
→ 使用3次「闹钟」挂机收益（10秒）
→ 刷技能本「颜料保卫战」耗体力（1-2分钟）
→ 推「世界染色」关推进度（1-2分钟）
→ 完成版本活动任务（1分钟）
→ 兑换商店买金票/紫票（10秒）
→ 快速放置收益买满（10秒）
→ 下线
```

**5分钟的爽点设计：**

- **收菜爽**：挂机收益一键领取，资源哗啦啦进账
- **推进爽**：每天稳定推几关，感受进度在涨
- **囤货爽**：每天稳定攒抽卡券，期待感拉满
- **碎片化**：5分钟搞定，完全不觉得是负担

**设计关键：** 日常耗时极短，但"收菜"这个动作本身有解压感，同时让玩家养成每天打开的习惯。

---

### ⏱ 30分钟 — 沉浸循环（深度体验）

> **场景：** 午休、通勤时间长、睡前躺床

**玩家行为流（多种选择）：**

```
【选择A：推主线/PVE】
连续刷关卡 → 卡关 → 检查弱点颜色 → 换阵容 → 再战
→ 每过一关的推进感 + 新功能解锁

【选择B：刷资源本】
颜料保卫战（技能本） × N轮
喷溅之旅（武器本） × N轮
→ 角色战力稳步提升

【选择C：联机对抗】
三人联机组队 → 分组对抗 / BOSS伤害PK
→ 社交互动 + 竞争快感

【选择D：研究阵容/养成】
查看角色图鉴 → 调整配饰装备 → 研究技能搭配
→ 策略深度带来的智力满足
```

**30分钟的爽点设计：**

| 行为 | 爽点 |
|------|------|
| 连续推关 | **突破爽**：卡了一天的关终于过了，多巴胺爆棚 |
| 刷本提升 | **变强爽**：技能升级/装备强化后战力数字飙升 |
| 联机对抗 | **社交爽**：和朋友互坑，物理超度对手 |
| 研究阵容 | **策略爽**：找到了最优配队，成就感 |
| 收集角色 | **收集爽**：抽到新角色，解锁图鉴 |

## 三、核心驱动循环图

```
                  ┌──────────────────────────┐
                  │      90秒一局爽一次       │
                  │  (乱涂爆发 + 弱点击破)     │
                  └───────────┬──────────────┘
                              │
                              ▼
                  ┌──────────────────────────┐
                  │     养成系统推进成长      │
                  │  (角色升级/技能/装备/共鸣) │
                  └───────────┬──────────────┘
                              │
                              ▼
                  ┌──────────────────────────┐
                  │   解锁新内容/新功能     │
                  │  (新关卡/新角色/新模式)   │
                  └───────────┬──────────────┘
                              │
                              ▼
                  ┌──────────────────────────┐
                  │    更多资源 → 更强角色    │
                  │  (挂机收益 + 日常产出)    │
                  └───────────┬──────────────┘
                              │
                              ▼
                    ┌──────────────────┐
                    │   回到底层战斗    │
                    │ (更强 → 更爽)    │
                    └──────────────────┘
```

**核心逻辑：** 战斗爽 → 变强 → 解锁更多 → 获得更多资源 → 更强 → 战斗更爽

## 四、游戏系统全拆解（换皮参考架构）

### 1. 战斗系统

| 组件 | 描述 | 换皮建议 |
|------|------|---------|
| **核心动作** | 单指走位，自动攻击/染色 | 保持单手操作，降低门槛 |
| **乱涂等级** | 攻击累积等级→Lv5爆发模式，15%伤害+30%攻速移速 | 换皮：改成"充能""蓄力""狂热"等机制名 |
| **颜色标记** | 红/黄/紫/蓝/绿5种颜色，各带不同元素效果（火种/感电/腐蚀/冰冻/缠绕） | 可自由设计属性体系，策略深度就在这 |
| **弱点击破** | BOSS有弱点颜色，用对应颜色攻击累计击破条→BOSS僵直 | 核心策略机制，保留 |
| **自动瞄准** | 自动锁定敌人，玩家只需走位 | 必须保留，这是"副副副游"的核心 |

### 2. 养成系统

| 组件 | 设计 | 亮点 |
|------|------|------|
| **共鸣等级** | 升级卡槽而非单个角色，换角色不影响等级 | ✅ **零焦虑养成**，可放心拉满 |
| **装备共享** | 装备可自由切换，换角色无需重养 | ✅ 降低切换成本 |
| **角色升阶** | 巅峰+ → 神话+ 等阶位，解锁新能力 | 长期留存锚点 |
| **技能升级** | 训练卡升级技能，资源从技能本产出 | 常规设计 |
| **配饰系统** | 主词条+套装效果，类似圣遗物/御魂 | 可保留或简化 |

### 3. 资源系统

| 资源 | 产出途径 | 核心用途 |
|------|---------|---------|
| **钻石** | 邮件/任务/活动/充值 | 抽卡（UP池）、快速放置 |
| **金票/紫票** | 活动/商店兑换/任务 | 抽卡 |
| **金币** | 挂机/关卡/日常 | 角色升级、基础强化 |
| **技能训练卡** | 颜料保卫战（技能本） | 技能升级 |
| **武器养成材料** | 喷溅之旅（武器本） | 装备强化 |
| **经典软膏** | 分解装备/挂机奖励 | 装备锻造 |

### 4. 挂机系统

| 组件 | 描述 |
|------|------|
| **世界染色** | 主线挂机关，进度越高挂机收益越高 |
| **24小时自动推图** | 神明娘自动爬塔，离线资源储备 |
| **闹钟道具** | 每天3次，直接领取2小时挂机收益 |
| **快速放置** | 钻石买满，性价比极高 |

### 5. 社交系统

| 组件 | 描述 |
|------|------|
| **三人联机** | 分组对抗 / 甜蜜三排 / BOSS伤害PK |
| **社团（公会）** | 社团商店兑换角色（伊邪那美） |
| **对抗互坑** | 可物理超度对手，增加趣味性 |

## 五、爽点类型总表（换皮核心参考）

| 爽点类型 | 触发场景 | 设计要点 |
|---------|---------|---------|
| 🎨 **视觉爽** | 颜料喷溅、颜色覆盖、全屏爆发 | 画面特效要够炫，屏幕要"爆" |
| 📈 **成长爽** | 乱涂等级递增，伤害数字变大 | 实时反馈，数字变化要明显 |
| 💥 **爆发爽** | Lv5乱涂爆发模式 | 要有明显的视觉和音效高潮信号 |
| 🎯 **策略爽** | 颜色克制、弱点击破 | BOSS弱点设计要直观可见 |
| 🏆 **收获爽** | 战斗结算、掉落奖励 | 结算动画要有仪式感 |
| ☕ **收菜爽** | 挂机收益一键领取 | 收益数字要让人舒服 |
| 🃏 **收集爽** | 抽卡、角色图鉴 | 角色设计要有魅力，抽卡动画要爽 |
| 🤝 **社交爽** | 联机互坑、组队 | 互动要有趣味性，不是纯竞技压力 |

## 六、换皮建议总结

### 核心保留（别改）
1. **单指操作 + 自动攻击** — 易上手是根本
2. **短对局（90秒）** — 碎片化的灵魂
3. **累积爆发机制**（乱涂等级→爆发模式）— 每局都有高潮
4. **属性克制 + 弱点击破** — 策略深度的核心
5. **挂机收益** — 留存利器，让玩家每天想上线收菜
6. **零焦虑养成**（共鸣等级/装备共享）— 敢拉满，不怕浪费

### 可换皮的要素
| 原版 | 换皮方向举例 |
|------|------------|
| 涂色/染色 | 泼墨/喷漆/画画/填色/涂鸦 |
| 神明娘 | 精灵/动物/机甲/国风仙子/像素角色 |
| 外星人入侵 | 病毒入侵/黑暗吞噬/墨水污染 |
| 颜色体系 | 五行/元素/星座/魔法学派 |
| Lv5爆发 | 暴走/觉醒/合体/超频 |

## 七、一句话总结

> **《乱涂彩世界》= 涂色皮的单手弹幕射击 + 卡牌养成 + 三人社交**
> 
> **爽而短的战斗（90秒）是钩子，零焦虑的养成是绳子，挂机收菜是胶水。**
