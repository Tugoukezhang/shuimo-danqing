# 水墨丹青·泼墨修复

> 东方水墨风弹幕射击游戏 — 基于《乱涂彩世界》核心玩法的换皮开发项目

## 项目简介

上古水墨长卷被墨魔侵蚀，画灵泼墨还原古画世界。

五色体系：朱砂 / 石绿 / 藤黄 / 花青 / 紫（对应国画颜料五色）

核心玩法：涂色弹幕射击 + 卡牌养成，90秒一局高速战斗节奏。

## 技术栈

- **引擎**: Canvas 2D (纯原生 JavaScript，无框架依赖)
- **方向**: Web / 微信小游戏
- **代码**: `demo/game.js` (4772行) + `demo/index.html`

## 运行方式

```bash
cd demo
# 用任意静态服务器打开 index.html
# 例: python -m http.server 8080
# 然后访问 http://localhost:8080
```

## 已实现功能 (Phase 1-3)

### 核心战斗
- 虚拟摇杆移动 + 自动瞄准 + 自动射击
- 五色循环弹丸 + 颜色护甲 + 弱点击破
- 闪避/冲刺（双击冲刺 + 无敌帧）
- 元素反应（颜色标记碰撞→爆炸+大范围涂色）
- 墨意爆发（怒气）系统
- BOSS击破条 + 三阶段AI + 多技能（十字墨爆/墨雨倾盆）
- 顿帧 + 震屏 + 弹性动画 + 6种音效
- 涂色画布（独立Canvas持久涂色 + 爆发晕染清空）

### 地图与关卡 (Phase 2)
- 大地图系统 (1200×1600 → 1600×2000) + 摄像机跟随
- 3个关卡（山水篇/花鸟篇/人物篇）+ 差异化背景装饰
- 7种障碍物 + 碰撞系统
- 关卡选择界面 + 存档系统 + 星级评价
- 标题画面 + 结算画面 + 暂停功能

### 角色与养成 (Phase 3)
- 3个画灵角色：青岚(均衡/回血) / 丹霞(暴击/溅墨) / 墨痕(范围/墨影)
- 共鸣等级系统（全角色共享养成，杀敌经验，升级+全属性）
- 爆发技能 + 被动技能（每个画灵独立）
- 画灵选择画面
