# 🎮 核心战斗系统 — 逐模块技术拆解

> 基于原版《乱涂彩世界》逆向拆解 + 玩家反馈 + 改进方案  
> 用途：写代码前的需求文档

## 模块一：触控移动系统

### 原版方案

| 维度 | 细节 |
|------|------|
| **视角** | 2.5D 竖屏，俯角约 45° |
| **控制方式** | 虚拟摇杆（Virtual Joystick），左手拇指操作 |
| **交互细节** | 单指在屏幕左半区滑动 → 角色向对应方向移动 |
| **移动逻辑** | 摇杆偏移量 → 归一化方向向量 → 角色恒定移速移动 |
| **转向** | 角色面向 = 移动方向，攻击方向 = 角色面向 |
| **移速** | 固定值无加速（玩家反馈"走位便秘"的原因之一） |
| **边界** | 关卡有边界碰撞，不可出屏 |

### 手感问题（来自玩家反馈）

> "移动手感有点便秘"——玩家原话  
> 没有闪避/冲刺键  
> 近战想极限躲技能但没有机动手段

### 换皮改进方案

```
移动输入管道
┌─────────────────────┐
│ Touch Start          │ 手指按下
├─────────────────────┤
│ Touch Move (Delta)   │ 手指滑动 → 计算偏移量
├─────────────────────┤
│ Normalize Vector     │ 归一化为单位方向向量
├─────────────────────┤
│ Apply Movement       │ 角色以 baseSpeed × deltaTime 移动
├─────────────────────┤
│ Update Facing        │ 角色面向 = 移动方向（松开恢复默认朝向）
└─────────────────────┘

改进点：
  1. Dashing（闪避）
     - 快速朝同一方向滑动两次 → 短距离冲刺
     - 冷却 2-3 秒 → 解决"近战便秘"问题
  2. 移速增益途径
     - 战斗中掉落"加速颜料" → 临时提升移速30%
     - 爆发期间移速自然提升
  3. 碰撞优化
     - 角色与敌人之间做 Pushback（受击推开）
     - 避免角色被小兵"卡位"堵死
```

## 模块二：自动攻击与瞄准系统

### 原版方案

| 维度 | 细节 |
|------|------|
| **攻击触发** | **全自动**——无需按攻击键，角色到达射程后自动开火 |
| **瞄准目标** | 自动锁定**最近的敌方单位** |
| **攻击方向** | 角色面向方向，跟随移动方向变化 |
| **攻击频率** | 取决于角色的攻速属性（Attack Speed） |
| **攻击类型** | 职业决定：近战扫射 / 射击连发 / 魔法弹 / 炮击AOE |
| **技能释放** | 右侧技能按钮，手动点击释放 |
| **大招QTE** | 三切大招：切换角色→释放大招→切下一个→累积连携 |
| **职业差异** | 近战：自动朝面向扇形AOE / 射击：自动锁定目标连射 |
|               | 魔法：自动蓄力后释放技能弹 / 炮击：自动朝目标方向范围轰击 |

### 存在的问题

- 自动攻击 + 自动瞄准 → **玩家只需走位**，操作深度有限
- 远程角色始终自动锁定最近的敌人，不能手动选择目标
- 没有"停止攻击"的选项，拉怪时可能提前触发战斗

### 换皮改进方案

```
攻击系统管道
┌─────────────────────────────────┐
│ Perception（感知阶段）            │
│ 扫描周围 → 获取可攻击目标列表      │
│ 按优先级排序：Boss > 精英 > 小兵   │
├─────────────────────────────────┤
│ Lock-on（瞄准阶段）               │
│ 锁定优先级最高的目标               │
│ 角色面向对准目标（平滑旋转）        │
├─────────────────────────────────┤
│ Attack（攻击阶段）                │
│ 判断是否在射程内 → 是则触发攻击     │
│ 生成弹丸 / 近战判定范围             │
│ 攻速冷却控制（每一发有间隔）        │
├─────────────────────────────────┤
│ Attack Pattern（职业差异）         │
│ 近战：扇形范围判定，AOE扫射         │
│ 射击：向目标方向发射单发子弹         │
│ 魔法：蓄力 → 释放技能弹 → CD       │
│ 炮击：向目标区域发射范围弹丸         │
└─────────────────────────────────┘

改进点：
  1. 加入手动切换目标
     - 点击敌人 = 锁定该目标（优先级最高）
     - 不操作时自动选最近的
  2. 加入"停火"状态
     - 移动时朝反方向拖拽 = 停止攻击（用于拉怪）
  3. 自动射击保留，但加入攻击方向微调
     - 右侧拖拽可微调射击方向
```

## 模块三：敌人AI系统

### 原版敌人行为

#### 普通小兵

| 行为 | 细节 |
|------|------|
| **移动** | 向玩家方向移动，无复杂路径规划 |
| **攻击** | 到达射程后发射弹幕/近战攻击 |
| **特殊** | 部分有颜色护甲/护盾（对应颜色攻击才能破） |
| **生成** | 关卡预设波次（Wave），清完一波出下一波 |

#### 精英怪

| 行为 | 细节 |
|------|------|
| **颜色护甲** | 绿色护甲（红色克）、黄色护盾（蓝色克） |
| **攻击** | 更复杂的弹幕，附带追踪属性 |
| **标记** | 身上有明显颜色标识，提示弱点 |

#### BOSS（以八岐酷蛇为例）

| 维度 | 细节 |
|------|------|
| **技能** | 6个技能：十字爆弹/毒液球/钻地蛇/激光蛇/毒蛇召唤/全屏引爆 |
| **阶段** | 单一阶段，血量越低技能释放越频繁 |
| **弱点** | 红/黄/紫三种颜色弱点，需踩光暗圈切换属性 |
| **特殊** | 召唤毒蛇小怪、光暗圈机制 |

### 敌人AI有限状态机

```
Enemy AI FSM
┌─────────────────────────────────┐
│ IDLE（待机）                      │
│ → 生成后进入，延迟1-2秒后激活      │
├─────────────────────────────────┤
│ CHASE（追击）                     │
│ → 以 speed 向玩家位置移动          │
│ → 到达 attackRange 后切换至 ATTACK │
├─────────────────────────────────┤
│ ATTACK（攻击）                    │
│ → 发射弹幕 / 近战攻击              │
│ → 攻击后进入 COOLDOWN（短暂后摇）  │
│ → 如果超出射程，切回 CHASE         │
├─────────────────────────────────┤
│ STUN（受击僵直）                   │
│ → 被弱点击破时触发                │
│ → 持续 2-3 秒，不受控制且易伤      │
├─────────────────────────────────┤
│ SPECIAL（特殊行为）                │
│ → 精英/BOSS独有                  │
│ → 召唤小怪 / 全屏技能 / 属性切换   │
└─────────────────────────────────┘
```

### 换皮改进

| 原版问题 | 改进方案 |
|---------|---------|
| 小兵只会傻追→射 | **加入基础走位**：小兵呈扇形散开包围玩家 |
| 没有仇恨机制 | **加入仇恨值系统**：坦克角色 / 高伤害角色吸引仇恨 |
| 精英怪只有颜色护甲 | 加入**可破坏部位**（打掉=短暂瘫痪） |
| BOSS技能不够多样化 | BOSS加入**阶段切换**：血量到75%/50%/25%时转阶段 |

## 模块四：射击命中视觉系统

### 原版方案

| 视觉元素 | 细节 |
|---------|------|
| **子弹表现** | 颜色涂鸦风格弹丸，与角色颜色一致 |
| **命中效果** | 颜料喷溅（Color Splash），被击中区域短暂染色 |
| **颜色标记** | 敌人身上叠加颜色标记（红喷喷/黄闪闪等），可见气泡图标 |
| **弱点击破** | BOSS血条下方白色进度条填充，满时顿帧+碎屏特效 |
| **爆发模式** | Lv5时角色周身特效变化+屏幕色调变化 |
| **屏幕染色** | 攻击命中时屏幕边缘有颜料溅射覆盖效果 |
| **颜色反应** | 两种颜色标记碰撞 → 爆炸特效 + 大范围染色 |
| **大招QTE** | 全屏特写画面，角色动画演出 |

### 存在的问题

> "战斗吸引了玩家9成以上的视觉精力"——玩家原话  
> 弹幕密集时看不清自己和敌人的子弹（颜色区分度不足）  
> buff/debuff状态没有有效反馈

## 模块六：战斗循环模拟

### 一场标准战斗的完整时间线（约90秒）

```
Phase 0: 关卡加载（2秒）
├── 显示关卡信息 + 弱点颜色提示
└── 进入战斗按钮

Phase 1: 小怪波次（15-20秒）
├── 生成3波小兵，每波4-6只
├── 玩家走位 → 自动攻击
├── 乱涂等级从 Lv0 → Lv3（累积中）
├── 颜色标记叠加在小兵身上
└── 清完小兵后过渡到BOSS

Phase 2: BOSS登场（5秒）
├── BOSS入场动画（无敌状态）
├── 屏幕震动
├── BOSS血条出现（下方弱点击破条）
└── 正式开始战斗

Phase 3: BOSS战斗 - 第一阶段（20-30秒）
├── BOSS使用基础技能（十字爆弹/毒液球）
├── 玩家走位躲避弹幕
├── 踩颜色圈切换攻击属性
├── 弱点击破条缓慢累积
└── 乱涂等级达Lv4

Phase 4: 爆发阶段（15-20秒）
├── 乱涂等级达 Lv5 → 爆发模式启动
├── 角色周身发光，攻速/伤害暴增
├── 弱点击破条快速累积
└── 注意躲避BOSS特殊技能

Phase 5: 弱点击破窗口（10-15秒）
├── 弱点击破条满 → BOSS进入僵直
├── 顿帧+慢动作特效
├── BOSS 2-3秒内无法行动，伤害大幅提升
├── 此时触发颜色反应 → 最高伤害窗口 (Double Dip)
└── BOSS恢复 + 召唤毒蛇小怪

Phase 6: BOSS战斗 - 后期（15-20秒）
├── BOSS技能频率加快
├── 释放全屏引爆（最致命）
├── 玩家躲到安全区/使用位移
├── 清掉召唤的小怪
└── BOSS血量见底

Phase 7: 结算（3秒）
├── BOSS击杀 → 慢动作终结画面
├── 经验值/掉落物结算
├── 三星评价判定
└── 关卡完成
```
