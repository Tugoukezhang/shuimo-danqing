## 🔴 P0 — 核心战斗资源（必须先行制作）

### 1. 玩家角色：画灵

| 项目 | 内容 |
|------|------|
| **设计定位** | 从古画中苏醒的水墨精灵，飘逸灵动，带书卷气 |
| **风格** | 写意水墨风，Q版（头身比约1:2），二次元与国风融合 |
| **色调** | 主色：墨黑 + 留白，点缀色：朱砂红（衣带/饰品） |
| **尺寸** | 站立帧：~80×100px，战斗帧：~100×100px（含攻击动作外扩） |
| **关键特征** | 宽袍大袖、飘带/衣带、发髻或书生帽、腰间挂笔或砚台 |
| **动画需求** | Idle（身体微起伏）、Run（衣袂飘动）、Attack（出招动作）、Hurt（受击后仰） |
| **参考意象** | 泼墨仙人图（梁楷）、敦煌飞天飘带、水没都市风 |

**设计提示词：**
> 一张游戏角色立绘，Q版中国水墨画风格角色，身高约头身比1:2。角色是古画中觉醒的水墨精灵——书生/仙子形象，宽袍大袖，飘逸墨色长袍，腰间挂一支毛笔。整体以墨黑色和宣纸白为主色，衣带用朱砂红色点缀。面部留白写意，头发化作墨迹飞扬。背景透明。角色气质：清新、飘逸、带几分书卷气的调皮。

| 分镜 | 提示词补充 |
|:----:|-----------|
| **Idle** | 角色站立，身体轻微上下浮动，衣带随风微动，宛如墨在水中晕开。 |
| **Run** | 角色向前奔跑，宽袍大袖向后飘扬，衣带化作墨迹拖尾。 |
| **Attack** | 角色挥手甩出墨点，指尖残留墨痕，动作一气呵成如书法挥毫。 |
| **Hurt** | 角色被击中后仰，身体化作墨迹散开再恢复。 |

### 2. 墨点子弹

| 项目 | 内容 |
|------|------|
| **设计定位** | 射出的墨点/墨滴，不是实心圆，要有笔触感 |
| **风格** | 水墨写意，墨滴飞溅 |
| **尺寸** | ~16×16px（动画帧内） |
| **形态** | 不规则滴水状，边缘有墨渣飞溅感，不是完美圆形 |
| **动画需求** | 飞行时拖尾（速度线）+ 尾部细墨点飘散 |
| **颜色变化** | 普通：墨黑色 / 暴击：朱砂红色 |

**设计提示词：**
> 一滴飞溅的墨滴，中国水墨画风格，形状不规则（非完美圆形），边缘有墨渣飞溅效果。墨滴中间浓黑（#1a1a1a），边缘半透明晕开。背景透明。适合用作游戏子弹精灵。约16×16像素。

---

### 3. 敌人：墨傀

| 项目 | 内容 |
|------|------|
| **设计定位** | 被墨魔侵蚀后失控的水墨造物，外形扭曲 |
| **风格** | 残破的水墨人偶/纸人，边缘墨迹渗开 |
| **尺寸** | 站立帧：~70×90px |
| **色调** | 主色：灰墨色（#555），夹杂污浊的墨渍 |
| **动画需求** | Idle（摇晃）、Hurt（墨迹震荡）、Death（散成一滩墨） |
| **视觉提示** | 身上有颜色弱点标记（朱砂红点/石绿纹等），提示玩家属性克制 |

**设计提示词：**
> 一个扭曲的墨色人形生物，中国水墨画风格。形状像被墨水污染撕碎的纸人，身体残缺不全，边缘墨迹不断渗开。颜色为灰黑墨色（#555），身上有未被完全侵蚀的颜色残留（如朱砂红点或石绿条纹）。没有任何面部细节——面部是一团漩涡状的墨。背景透明。Q版比例，约70×90像素。

## 🟡 P1 — 视觉增强资源（Demo跑通后再替换）

### 4. 墨水喷溅粒子

当前使用Phaser内置的circle渐隐模拟，用tween实现。正式资源需要一套**水墨晕染序列帧**。

| 帧 | 描述 | 尺寸 |
|:--:|------|:----:|
| 1 | 墨点溅开（小） | 32×32 |
| 2 | 墨点溅开（中） | 48×48 |
| 3 | 墨点溅开（大） | 64×64 |
| 4 | 墨渍残留（地面） | 32×32~80×80 |

**设计提示词：**
> 中国水墨画的墨点溅开动画序列帧。黑色墨水在宣纸上由中心向外晕染扩散，边缘不规则，呈现出水渍浸染的自然效果。背景透明。每个约64×64像素，共4帧。

---

### 5. 伤害数字字体

当前使用系统字体 (`STKaiti`)。正式版本需配套**笔墨手写体数字**。

| 类型 | 描述 |
|:----:|------|
| 普通伤害 | 灰黑色正楷数字，大小约20px |
| 暴击伤害 | 朱砂红色行草数字，大小约28px，带墨迹拖尾 |

**建议方案：**
- 使用免费可商用毛笔字体：**站酷仓耳渔阳体**、**演示夏行楷**、**思源宋体**
- 写入游戏时用 `@font-face` 加载，不需要切图

### 6. 宣纸背景

| 项目 | 内容 |
|------|------|
| **当前** | Phaser程序化生成（渐变+半透明噪点+山形轮廓） |
| **替换目标** | 一张390×844px的完整水墨背景图 |
| **内容** | 宣纸纹理 + 远山/墨竹/飞鸟等写意元素 |
| **调性** | 留白为主，不要填满——中国画重在"留白" |

**设计提示词：**
> 中国水墨画风格的竖屏游戏背景。画面以大量留白为主，底色为宣纸色（#f5f0e8）。远处有极淡的墨色远山轮廓，山间飘着几缕薄雾。右上角有几株墨竹剪影。整体意境：空灵、写意、安静。竖屏比例390×844像素。色调饱和度极低。

## 🟢 P2 — UI资源（最后做）

### 7. 虚拟摇杆

| 项目 | 内容 |
|:----:|------|
| 当前 | 程序化半透明圆形 |
| 替换 | 水墨风格圆形：外圈为墨色笔触画圆，内圈淡墨晕染 |
| 尺寸 | 外圈~120×120px，内圈~48×48px |

**设计提示词：**
> 中国水墨画风格的双层圆形摇杆。外圈是毛笔一笔画出的不完全圆形墨线，有笔触的起笔和收笔尖峰。内圈是轻轻点出的墨点，边缘自然晕开。整体色调：墨黑色半透明。背景透明。外圈120×120px，内圈48×48px。

### 8. 战斗UI

| 元素 | 描述 |
|:----:|------|
| 墨意条 | 横向条状，左侧有毛笔笔触装饰，填充墨色渐变 |
| 击杀计数 | "破：N" 毛笔字样式 |
| 招式名 | 弹出"·泼墨·"等招式名时，大字毛笔效果 |
| 按钮 | 如后续加入技能按钮，做水墨卷边效果 |

## 技术备注

- 所有资源目标格式：**PNG（透明背景）**
- 角色动画：建议使用 **Spine** 或 **DragonBones** 骨骼动画，或Phaser **Sprite Sheet**（序列帧）
- 粒子效果：Phaser自带 Particle Emitter，或自定义 Sprite 动画
- 字体：用 `@font-face` 加载 `.ttf/.woff2` 格式的毛笔字体
